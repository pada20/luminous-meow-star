# 🌟 Luminous Meow Star

歡迎來到 **Luminous Meow Star**！這是喵喵的個人網站，致力於語言學習、故事創作與角色扮演的全方位平台。

## 🐱 專案簡介

本專案旨在打造一個多語言學習平台，結合語音控制、故事創作與角色扮演功能，讓學習變得有趣且互動性十足。

## 🔗 線上預覽

👉 [點我查看網站](https://pada20.github.io/luminous-meow-star/)

## 📂 功能特色

- 🎤 語音控制開關
- 🗣️ 多語言學習模組（含法文、泰文等）
- 📖 故事與角色構思工具
- ✍️ 文案撰寫輔助
- 🎨 多種風格主題切換
- 🛠️ 自定義功能擴充能力
